# System Design — Thiết kế màn hình

Tài liệu mô tả wireframe và luồng tương tác của từng màn hình trong ứng dụng Finance Web.

---

## Bố cục chung (App Layout)

Tất cả màn hình sau khi đăng nhập đều dùng layout 2 cột: Sidebar trái cố định + Content area phải.

```
┌─────────────────────────────────────────────────────────────────────┐
│  TOPBAR                                              [User] [Logout] │
├────────────┬────────────────────────────────────────────────────────┤
│            │                                                         │
│  SIDEBAR   │                  CONTENT AREA                          │
│            │                                                         │
│  Dashboard │                                                         │
│  Tài khoản │                                                         │
│  Chuyển    │                                                         │
│  tiền      │                                                         │
│  Vay vốn   │                                                         │
│  Báo cáo   │                                                         │
│            │                                                         │
│ ─────────  │                                                         │
│  [ADMIN]   │                                                         │
│  Quản lý   │                                                         │
│  tài khoản │                                                         │
│  Giao dịch │                                                         │
│  Audit Log │                                                         │
│  Gian lận  │                                                         │
│            │                                                         │
└────────────┴────────────────────────────────────────────────────────┘
```

> Menu Admin chỉ hiển thị khi `user.roles` chứa `ADMIN`.

**Trạng thái:**
- `idle` → form chờ nhập
- `loading` → nút hiện spinner, disabled
- `error` → hiện thông báo lỗi dưới form (vd: "Sai số tài khoản hoặc mật khẩu")
- `success` → redirect sang `/dashboard`
